# 题解

## 思路
```text
1. 将数据根据值，数组位置，存储到一个 map 中，map 的结构是 value - list<locations>
2. 新建另外一个标志 map，用来记录当前的位置是否满足条件要求
3. 遍历第一个 map
    * 如果 map 的值的第一个数据就是位置已经满足要求，则下一条
    * 如果不满足，说明当前值未经过验证，则通过遍历 30 次 2 的幂次方来计算 当前值与幂次方的差 dis
    * 如果 dis 小于 0 或 dis 在 map 中不存在 则下一次 2 的幂次方循环
      * 如果 dis 与当前值相同，且当前值的 list size 为 1 则继续当前 map 循环，此数不满足要求
      * 剩下的就是满足条件的数据，通过遍历当前以及 dis 的 map 的 list 操作 标记 map
4. 返回 n - 标记 map 的大小
```

## 注意
```text
1000000000 < 2 的 30 次幂
```
